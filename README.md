# Sharpe EduQuality Connect Phase 1 MVP

This is a simple browser prototype for the first version of Sharpe EduQuality Connect.

## How to Open

Double-click `index.html` in this folder:

`C:\Users\gurne\Documents\Codex\2026-05-28\proposed-framework-for-the-college-quality\index.html`

## Included Features

- Home dashboard
- Notices and announcements
- Surveys and evaluations
- Feedback, complaints, and suggestions
- Policy and resource library
- QualityBot chatbot prototype
- Basic admin dashboard
- Basic installable web app support for devices

## Notes

This version stores sample notices and feedback tickets in the browser using `localStorage`.
It is a prototype, so it does not yet connect to Firebase, OpenAI, Microsoft login, or a real document database.

For phone or tablet installation, host the folder online first. Then open the link on the device and use the browser option to add the app to the home screen.
