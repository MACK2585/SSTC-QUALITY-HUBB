# Upload Sharpe EduQuality Connect to GitHub

## Option 1: Upload Through the GitHub Website

1. Go to `https://github.com`.
2. Sign in to your GitHub account.
3. Click **New repository**.
4. Repository name suggestion:

   `sharpe-eduquality-connect`

5. Choose **Public** or **Private**.
6. Click **Create repository**.
7. Click **uploading an existing file**.
8. Upload these files:

   - `index.html`
   - `styles.css`
   - `app.js`
   - `manifest.webmanifest`
   - `service-worker.js`
   - `icon.svg`
   - `netlify.toml`
   - `README.md`
   - `START-HERE.txt`
   - `.gitignore`

9. Click **Commit changes**.

## Option 2: Use the GitHub Upload Zip

Upload the contents of:

`Sharpe-EduQuality-Connect-GitHub-Upload.zip`

Do not upload the older prototype zip files into GitHub unless you specifically want archived copies.

## After Upload

You can publish it using GitHub Pages:

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Click **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and `/root`.
6. Save.

GitHub will create a website link for the app.
